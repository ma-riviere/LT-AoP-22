Chosen model: Gamma (‘log’) + AR1 
```{r}
glmmTMB(formula = Weight ~ Condition * Stage + ar1(Stage + 0 | 
    Mouse), data = data, family = Gamma("log"), REML = TRUE, 
    ziformula = ~0, dispformula = ~1)
```
